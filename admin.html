คู่มือการใช้งาน Admin Panel

📌 ภาพรวม: Admin Panel จัดการเนื้อหาทั้งหมดของเว็บไซต์ห้องเรียนออนไลน์ ตั้งแต่ห้องเรียน รายชื่อนักเรียน ประกาศ ใบงาน ข้อสอบ วิดีโอ ผลสอบ ไปจนถึงแชทและการตั้งค่าเว็บ
1🏫 ห้องเรียน — จัดการชั้นเรียนและรายชื่อนักเรียน
  1. กดปุ่ม ➕ เพิ่มห้อง → กรอกชื่อห้อง (เช่น ป.5, ม.3/1) และรหัสห้อง
  2. กดปุ่ม 👥 รายชื่อ ข้างชื่อห้อง → เพิ่ม/ลบชื่อนักเรียนในห้องนั้น
  3. ชื่อนักเรียนที่เพิ่มที่นี่ จะปรากฏในดรอปดาวน์หน้า Login ของนักเรียนโดยอัตโนมัติ
⚠️ ชื่อห้องเรียนต้องตรงกับชื่อชั้นในระบบเก็บคะแนน (scores.html) เช่น ป.5 หรือ ม.3 เพื่อให้รายชื่อโหลดได้ถูกต้อง
2📢 ประกาศ — แจ้งข่าวสารและมอบหมายงาน
  1. กด ➕ เพิ่มประกาศ → กรอกหัวข้อ เนื้อหา เลือกห้อง และวิชา
  2. ใส่ URL รูปภาพ (ไม่บังคับ) เพื่อแนบรูปประกอบ
  3. เลือก ทุกห้องเรียน เพื่อส่งประกาศถึงทุกชั้นพร้อมกัน
  4. กด ✏️ แก้ไข หรือ 🗑️ ลบ ที่ท้ายแถวได้เลย
3📄 ใบงาน — แนบเนื้อหาและไฟล์สำหรับนักเรียน
  1. กด ➕ เพิ่มใบงาน → กรอกชื่อใบงาน วิชา และคำอธิบาย
  2. ลิงก์ไฟล์: ใส่ URL Google Drive (ตั้งให้ใครก็ดูได้ก่อน) หรือลิงก์อื่นๆ
  3. HTML Content: วางโค้ด HTML ใบงานที่สร้างโดย Claude ได้โดยตรง
💡 สร้างใบงาน HTML ด้วย Claude แล้ววางในช่อง HTML Content — นักเรียนเปิดอ่านได้ทันทีโดยไม่ต้องดาวน์โหลด
4📝 ข้อสอบ — สร้างและจัดการข้อสอบ
มี 2 ประเภท:
  • ปรนัย (ตัวเลือก) — สร้างคำถาม+ตัวเลือกในระบบ ระบบตรวจให้อัตโนมัติ
  • ไฟล์ HTML — ใส่ URL ไฟล์ข้อสอบ HTML ที่อัปโหลดบน GitHub Pages นักเรียนกดแล้วเปิดไฟล์นั้น คำตอบส่งกลับมาให้ครูตรวจในหน้า ผลสอบ
  1. กด ➕ สร้างข้อสอบ → เลือกประเภท กรอกรายละเอียด กำหนดห้อง/วิชา
  2. สำหรับประเภทปรนัย: กด ➕ เพิ่มคำถาม เพื่อเพิ่มทีละข้อ กดวงกลมหน้าตัวเลือกเพื่อเลือกเฉลย
⚠️ ไฟล์ข้อสอบ HTML ต้องอัปโหลดขึ้น GitHub Pages ก่อน แล้วค่อยนำ URL มาใส่ในระบบ
5📊 ผลสอบ — ตรวจ ให้คะแนน และลบผลสอบ
  1. ดูรายการที่นักเรียนส่งทั้งหมด กรองได้ตามข้อสอบ ชั้น หรือวิชา
  2. กดปุ่ม ✏️ ตรวจ/ให้คะแนน เพื่อดูคำตอบรายข้อ แล้วกรอก คะแนนที่ได้ / คะแนนเต็ม
  3. กด 💾 บันทึกคะแนน → คะแนนจะแสดงในหน้าคะแนนของนักเรียนทันที
  4. กดปุ่ม 🗑️ ลบ เพื่อลบผลสอบที่ส่งมา (นักเรียนสามารถส่งใหม่ได้)
💡 ผลสอบที่ตรวจแล้วจะมีป้าย ✅ ตรวจแล้ว ผลสอบที่ยังไม่ตรวจจะมีป้าย ⏳ รอตรวจ
6💬 แชทนักเรียน — ตอบคำถามนักเรียน
  1. เลือกห้องเรียนที่ต้องการจากแถบปุ่มด้านบน
  2. กรอกข้อความในช่องด้านล่าง กด Enter หรือปุ่มส่ง
  3. ถ้ามีข้อความใหม่จากนักเรียน จะมี 🔴 แจ้งเตือนที่เมนู แชทนักเรียน และตัวเลขบนปุ่มห้องนั้น
⚠️ ระบบจะล้างแชทอัตโนมัติทุก 2 เดือน วันที่ 1 เพื่อประหยัดพื้นที่ Firestore
7🎬 วิดีโอ — แนบคลิปการเรียน
  1. กด ➕ เพิ่มวิดีโอ → ใส่ชื่อและ URL ของ YouTube
  2. ระบบจะแปลง URL เป็น embed อัตโนมัติ นักเรียนดูได้ในแอปเลยโดยไม่ต้องออกไป
8⚙️ ตั้งค่าเว็บไซต์
  • โลโก้โรงเรียน และ รูปโปรไฟล์ครู: ใส่ path เช่น images/logo.png หรือ URL รูปภาพออนไลน์
  • ชื่อโรงเรียน / ชื่อครู / ตำแหน่ง: อัปเดตแล้วบันทึก — ทุกหน้าจะอัปเดตทันที
  • กด 💾 บันทึกการตั้งค่า เพื่อบันทึกทุกการเปลี่ยนแปลงลง Firestore
💡 เปลี่ยนชื่อครูในตั้งค่า → ชื่อในแชท ส่วนหัว และทุกที่จะเปลี่ยนพร้อมกันอัตโนมัติ
9⭐ ระบบเก็บคะแนน (scores.html)
  • ระบบเก็บคะแนนรายวิชาแบบตารางสรุป — เพิ่ม/แก้ไขคะแนนรายคนรายวิชาได้โดยตรง
  • ข้อมูลซิงค์ขึ้น Firestore อัตโนมัติ ทุกอุปกรณ์เห็นคะแนนเดียวกัน
  • กด 🔙 Admin เพื่อกลับมาหน้านี้
⚠️ รายชื่อนักเรียนในระบบเก็บคะแนนจะถูกดึงมาใช้เป็นดรอปดาวน์ในหน้า Login ของนักเรียน (student.html) โดยอัตโนมัติ
